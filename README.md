## ✨ Features
- Aadhaar + OTP + DeepFace 2-factor auth
- Hyperledger Fabric blockchain vote ledger (simulated)
- zk-SNARKs voter anonymity + verifiability
- Cryptographic vote receipts
- Live results via smart contract auto-tallying
- Public audit portal
